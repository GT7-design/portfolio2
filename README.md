# 🏫 서울풍납초등학교 AI 웹툰 창작소 (포트폴리오 2)

이 프로젝트는 학교 현장에서 학생들의 창작 활동(국어, 창의적체험활동, 역사·지역화 등)을 지원하기 위해 개발된 **AI 웹툰 창작소** 교과 융합 산출물입니다. 학생이나 교사가 쓴 구체적인 스토리를 입력하면 선택한 컷 수에 맞게 장면을 자동으로 나누고, 각 컷별 요약 및 맞춤형 AI 일러스트를 생성하여 한 권의 책 형태로 감상할 수 있습니다.

---

## 📌 교사 개발자 성장형 산출물 조건 안내 충족 (01~08 항목)

본 산출물은 다음 8대 평가·심사 필수 조건을 100% 충족하도록 명확하고 체계적으로 설계되었습니다.

1. **01 소속학교·이름 표기:** 첫 화면(로그인), 상단 바, 하단 푸터, 메인 체크리스트 카드에 **[서울풍납초등학교 교사 김규태]** 소속과 이름을 명확히 표시했습니다.
2. **02 개인정보처리방침:** 수집 항목(구글 계정/시연 데이터), 이용 목적, 보관 기간, 문의처를 간단명료하고 직관적인 모달 및 카드 미리보기로 안내합니다.
3. **03 사용약관:** 서비스 이용 조건, 책임 및 저작권 범위, 금지 행위, 약관 변경 안내를 체계적으로 고지합니다.
4. **04 GitHub 레포지토리 주소:** 소스코드 공개 주소([GT7-design/portfolio2](https://github.com/GT7-design/portfolio2))를 제공하여 개발 이력과 협업 내용을 투명하게 확인할 수 있습니다.
5. **05 Firebase 백엔드:** Firebase Authentication(구글 인증) 및 Firestore Database를 실시간으로 연동하여 인증 및 산출물 데이터 저장을 안정적으로 처리합니다.
6. **06 프론트엔드 웹주소:** 로컬 실행뿐 아니라 실제 누구나 즉시 접속 가능한 Vercel 웹 배포 URL을 제공합니다.
7. **07 사용방법 안내:** 로그인 안내, 4단계 주요 창작 기능, 주의사항 및 팁을 직관적인 UI(상단 버튼 및 모달)로 정리해 처음 사용자도 1분 내에 이해할 수 있습니다.
8. **08 배포 완료 확인:** 웹 접속, 정책·약관 링크 작동, AI 이미지 실시간 렌더링, 책 보기 뷰어 및 데이터 저장 기능이 모두 정상 작동하도록 검증 완료했습니다.

## ⚡ 주요 핵심 기능

- **🟢 구글 계정 로그인 & ⚡ 시연용 즉시 로그인 동시 지원:**
  - Firebase Auth를 이용한 서울시교육청 계정(`@sen.go.kr`, `@school.kr` 등) 및 구글 계정 인증을 안전하게 지원합니다.
  - 심사위원 및 처음 방문자를 위해 구글 계정 없이도 **'⚡ 시연용 계정 즉시 로그인'** 버튼 한 번으로 테스트 계정(`demo@school.kr`)이 자동 입력되고 즉시 접속되어 모든 기능을 체험할 수 있습니다.
- **🖼️ 실시간 AI 웹툰 일러스트 생성 (`Pollinations AI` + `고품질 SVG 백업`):**
  - 입력된 스토리와 장르, 등장인물 정보를 바탕으로 각 컷에 가장 잘 어울리는 맞춤형 AI 영문 프롬프트를 자동 구성하여 실시간 AI 일러스트를 렌더링합니다.
  - 네트워크 지연이나 외부 API 응답 차질 시에도 깨진 이미지 없이 **자동으로 고품질 동적 SVG 웹툰 일러스트로 전환**되도록 이중 안전장치를 적용했습니다.
- **📖 책 형태로 감상하기 (`Storybook Flipbook Viewer`):**
  - 완성된 컷들과 이야기를 모아 단행본 전자책처럼 감상할 수 있습니다.
  - **[페이지 넘기기 모드]:** 표지부터 제 1장 ~ 제 N장까지 좌우 화살표로 페이지를 넘기며 컷 이미지, 스토리 본문, AI 연출 정보를 감상합니다.
  - **[전체 스크롤 모드]:** 상단 버튼을 통해 표지와 모든 장면을 한눈에 연속 스크롤하며 읽을 수 있습니다.
- **📚 클라우드 및 로컬 하이브리드 동기화:**
  - Firebase Firestore를 통해 로그인 계정의 생성 기록이 실시간 클라우드에 저장되며, 동시에 로컬 스토리지 하이브리드 관리로 시연 계정 환경에서도 저장 및 열람이 100% 정상 작동합니다.

## 🚀 실행 및 테스트 방법

이 프로젝트는 별도의 빌드 과정 없이 바로 실행 가능한 정적 웹 애플리케이션입니다.

1. 로컬 환경에서 VS Code Live Server 등 로컬 HTTP 서버로 `index.html`을 띄우거나, 배포된 Vercel URL에 접속합니다.
2. **로그인 체험:** 
   - `🟢 Google 계정으로 로그인` 또는 `⚡ 시연용 계정 즉시 로그인`을 선택하여 들어갑니다.
3. **웹툰 생성:** 
   - 작품 제목, 등장인물, 장르(역사 모험/교육 판타지 등), 컷 수(4/8/12컷)를 선택하고 스토리를 입력합니다.
   - `⚡ AI 웹툰 컷 및 일러스트 생성하기` 버튼을 누르면 약 5~10초간 스켈레톤 로딩이 진행된 후 고품질 컷 이미지가 나타납니다.
4. **책 보기 감상:** 
   - 생성 완료 후 나타나는 `📖 책 형태로 감상하기` 버튼을 눌러 표지 및 컷별 페이지를 넘기며 감상합니다.
